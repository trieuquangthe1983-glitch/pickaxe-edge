# Outreach asset package

The 30-day validation plan in [SALES_PLAYBOOK.md](../SALES_PLAYBOOK.md) requires
**30 conversations week 1**. This folder gives you the operating system to
execute that without reinventing the wheel each DM.

## The 30-30-3 rule

- **30 prospects** researched (week 1)
- **30 DMs** sent (week 1, batched 5/day to avoid platform throttling)
- **3 paid audits** closed (week 2)

If you have <30 DMs out by day 7, you don't have a product problem, you have
an outreach problem. Don't iterate the tool yet — iterate the message.

## Where to find prospects (free, no scraping needed)

1. **LinkedIn search:** "B2B SaaS founder" + filter posts last 7 days + sort by
   engagement. Anyone with 50+ reactions on a recent post is in your TAM.
2. **Twitter/X advanced search:** `from:* min_faves:200` + topic keyword.
   Niche-relevant posts with 200+ likes = active creator.
3. **YouTube:** type niche keyword + filter by upload date (this month) + sort
   by view count. Channels with 10k-100k subs are sweet spot.
4. **Substack:** browse https://substack.com/discover -> filter by topic. Newsletters
   with 1k-10k subscribers are sweet spot.
5. **Indie Hackers + Reddit r/Entrepreneur:** filter for "I just hit $X MRR"
   posts — they have cash and they're in the content marketing mindset.

## How much time this takes

- Research: 5 min/prospect x 30 = 2.5 hours total
- DM drafting (after first 5, you reuse the script): 2 min/DM x 30 = 1 hour
- Response handling (rolling): ~30 min/day for 2 weeks

Total commitment for full validation: **~10 hours over 14 days**.
